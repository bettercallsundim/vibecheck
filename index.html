interactive walkthrough

Your AI just shipped a plot twist.

One prompt later, your repo has new routes, helpers, env vars, and a migration with a suspiciously confident name. `vibecheck` turns that surprise diff into a quick field guide: where to start, what to skip, what might bite, and what to ask next.

Follow the story, not the file list

Diffs read like a receipt. Code runs like a story. VibeCheck orders the tour by how the feature actually works: data shape, logic, route, UI, tests.

Find the spicy parts fast

Auth touched? Payment webhook added? Env var renamed? Database rows deleted? VibeCheck marks the parts worth slowing down for, without yelling about every import.

Make your brain prove it loaded

Quiz mode asks three tiny, targeted questions about the exact diff. If you can answer them, you understand it. If not, you found the next thing to inspect.

Install once. Run after every agent session.

Use `npx` for a project install, add `--global` for every project on this machine, or copy the skill manually with curl.

npx - per project 
npx vibecheck-skill

Installs to `.agents/skills/vibecheck/SKILL.md` in the current project. Commit it to share with your team.

npx - global 
npx vibecheck-skill --global

Installs to `~/.agents/skills/` and `~/.claude/skills/` so it is available across projects.

curl - manual global 
mkdir -p ~/.agents/skills/vibecheck && \
curl -fsSL https://raw.githubusercontent.com/bettercallsundim/vibecheck/main/SKILL.md \
  -o ~/.agents/skills/vibecheck/SKILL.md

Restart your AI coding tool after any install method, then run `/vibecheck` in a repo with fresh changes.
